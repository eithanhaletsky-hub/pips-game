How to Play KenKen

Goal: Fill the grid with numbers so that:

  • Each row contains unique numbers from 1 to N (grid size)
  • Each column contains unique numbers from 1 to N
  • Numbers in each cage (thick-bordered group) combine using the shown operation to produce the target number

Cage Labels:

  • 12+ means the numbers in that cage add up to 12
  • 3- means the difference between the numbers is 3
  • 12× means the numbers multiply to 12
  • 2÷ means one number divided by the other is 2
  • A single number with no operation is a given (freebie)

Controls:

  • Click a cell and type a number (or use the number pad)
  • Toggle Pencil mode for small candidate notes
  • Toggle Errors to highlight mistakes
